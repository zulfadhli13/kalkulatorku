Luas Persegi

Rumus Luas Persegi Yaitu :

L = S x S

Rumus Keliling Persegi Yaitu :

K = 4 x S

  • Dimana :
  • L = Luas
  • S = Sisi
  • K = Keliling

- PERSEGI PANJANG -

Luas Persegi Panjang

Rumus Luas Persegi Panjang Yaitu :

L = p x l

Rumus Keliling Persegi Yaitu :

K = 2 x (p + l)

  • Dimana :
  • L = luas
  • S = sisi
  • K = keliling
  • p = panjang
  • l = lebar

Hasil
